# 時刻表示アプリ

100ミリ秒単位で現在時刻を全画面表示するWebアプリケーションです。

## 機能

- **リアルタイム時刻表示**: 100ミリ秒ごとに更新される高精度な時刻表示
- **全画面対応**: レスポンシブデザインでどのデバイスでも見やすい表示
- **フルスクリーン機能**: ダブルクリックでフルスクリーン表示に切り替え可能
- **美しいUI**: グラデーション背景とアニメーション効果

## 使い方

1. ページを開くと自動的に現在時刻の表示が開始されます
2. ダブルクリックでフルスクリーン表示に切り替えられます
3. ESCキーまたは再度ダブルクリックでフルスクリーンを終了できます

## 技術仕様

- **更新間隔**: 100ミリ秒
- **時刻精度**: ミリ秒単位（100ms刻み表示）
- **対応ブラウザ**: モダンブラウザ全般
- **レスポンシブ対応**: スマートフォン・タブレット・PC
